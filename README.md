Sistema Escolar em Python

Descrição

Sistema escolar desenvolvido em Python que funciona pelo terminal.
O programa permite cadastrar, listar, editar e remover alunos, além de calcular médias e situações.

Os dados são salvos no arquivo `dados.json`.

Funcionalidades

- Cadastrar aluno (nome, idade, turma e notas)
- Listar alunos cadastrados
- Editar informações dos alunos
- Remover alunos
- Listar notas, médias e situação:
  - Média >= 7: Aprovado
  - Média entre 5 e 6.9: Recuperação
  - Média < 5: Reprovado
